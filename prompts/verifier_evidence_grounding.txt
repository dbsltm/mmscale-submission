You are identifying the LOAD-BEARING evidence behind a model's moral verdict.

Input:
- An image
- A written scenario/action
- A model-generated reasoning trace ending in a 1-5 moral acceptability rating

By "load-bearing" we mean: evidence the verdict actually depends on. Decorative description of the image -- saying what is in it -- does NOT count as load-bearing if the verdict would be the same without that visual content. Only count visual evidence as load-bearing if it is plausibly the reason for the verdict.

Heuristic: ask yourself "if the image were replaced with a uniform gray patch (or replaced with a different image), would the verdict in this trace plausibly change?"
- If yes (the visual content is doing real work): image-grounded or both.
- If no (the image is described but the verdict would hold anyway): text-grounded.

Choose exactly one label:

1. text-grounded:
The verdict follows from the action description. The trace may describe the image, but the verdict would not change if the image were absent.

2. image-grounded:
The verdict follows from visual evidence. The action description (if mentioned) is not the load-bearing reason; the visual content is.

3. both:
The verdict requires combining text and visual evidence. Removing either modality would plausibly flip the conclusion.

4. ungrounded:
The verdict is asserted with no concrete grounding -- generic moral platitudes only, with no engagement with either the action description or the image content.

5. hallucinated-visual:
The trace cites visual content that is NOT present in the image (or contradicts it), AND that hallucinated content is presented as load-bearing for the verdict. If the trace cites real visual content but the verdict does not depend on it, do NOT use this label -- use text-grounded.

Important:
- Do not be misled by image description. A trace can describe the image at length and still be text-grounded if the verdict obviously follows from the action text alone.
- Concrete visual evidence must refer to observable entities, actions, settings, expressions, objects, spatial relations, or risks AND must be the cited reason for the verdict.
- Judge only the reasoning trace and the verdict it leads to, not whether the final moral verdict is correct.
